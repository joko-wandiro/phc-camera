== Description ==

Camera Slider make you create / add slider easily into your post / page / widget.
jQuery slideshow with many effects, transitions, easy to customize, using canvas and mobile ready, based on jQuery 1.4+. You can add it into post / page / widget Wordpress.

Features:
* Of Course it's easy to use.
* Good User Interface.
* WordPress 3.5 Media Uploader.
* Auto Regenerate Images.
* Translation available ( welcome for contributor ).
* Setting Page to set image size.
* Available Custom Hooks.
* Complete Documentation Page.

Any feedback will be appreciated to make this plugin better.

* License: [GNU GPL](http://www.gnu.org/licenses/gpl-3.0.txt)

== Installation ==

* Upload `plugin-name` to the `/wp-content/plugins/` directory
* Goto Plugins Menu then click Activate on Camera Slider Plugin.
